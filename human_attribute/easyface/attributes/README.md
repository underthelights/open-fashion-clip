# Facial Attribute Classification

## FairFace Dataset

* [WACV 2021 Paper](https://openaccess.thecvf.com/content/WACV2021/papers/Karkkainen_FairFace_Face_Attribute_Dataset_for_Balanced_Race_Gender_and_Age_WACV_2021_paper.pdf) 
* 3 attributes 
    * Gender: 2 classes (Male, Female)
    * Race: 7 classes (White, Black, Latino Hispanic, East, Southeast Asian, Indian, Middle Eastern)
    * Age: 9 classes (0-2, 3-9, 10-19, 20-29, 30-39, 40-49, 50-59, 60-69, 70+)
* 108K Faces
* Download the dataset from [here](https://drive.google.com/file/d/1Z1RqRo0_JiavaZw2yzZG6WETdZQ8qX86/view?usp=sharing).
* Download [train](https://drive.google.com/file/d/1i1L3Yqwaio7YSOCj7ftgk8ZZchPG7dmH/view?usp=sharing) and [validation](https://drive.google.com/file/d/1wOdja-ezstMEp81tX1a-EYkFebev4h7D/view?usp=sharing) annotation files.
* Face images are cropped and aligned.
